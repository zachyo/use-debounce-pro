# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-11

### Added

- `useDebouncePro` hook — debounce and throttle with a single API
- `useDebouncedCallback` hook — alternative named-methods API
- Leading and trailing edge support
- `maxWait` option for throttle behavior
- `cancel()`, `flush()`, `isPending()` control methods
- Full TypeScript support with generic type inference
- Zero runtime dependencies
- ESM and CJS dual-format builds
- 759 bytes gzipped
